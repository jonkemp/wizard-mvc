# wizard-mvc

> Wizard pattern for JavaScript MVC frameworks.

## Frameworks

* Backbone.js
* Backbone.js + Require.js
* Marionette.js
* Ember.js
* Angular.js

## Get Started

To get started, run `npm start` or `node server.js`. Your browser will automatically open to [http://localhost:3000/](http://localhost:3000/).
